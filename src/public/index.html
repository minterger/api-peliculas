Series

Obtener los Posters de Series
  • GET /series obtiene los posters de las series.
    Para navegar entre paginas usa el query ?page. ej: GET /series?page=3
  • GET /series/populares para traer los posters de las series populares.
    Para navegar entre paginas usa el query ?page. ej: GET /series/populares?page=3
  • GET /series/estrenos para traer los posters de las series estrenos.
    Para navegar entre paginas usa el query ?page. ej: GET /series/estrenos?page=3
Obtener informacion de una Serie
  • usa /serie/:info para traer la informacion de una serie. ej: GET /serie/the-flash
Obtener temporadas de una Serie
  • usa /serie/temporadas/:info para traer todas las temporadas de una serie. ej: GET /serie/temporadas/the-flash
Obtener reproductores de una Serie
  • usa /serie/:info/temporada/:temporada/capitulo/:capitulo para traer todas los reproductores de una serie y ver las opciones de navegacion entre capitulos.
    ej: GET /serie/the-flash/temporada/2/capitulo/16

Animes

Obtener los Posters de Animes
  • GET /animes obtiene los posters de las animes.
    Para navegar entre paginas usa el query ?page. ej: GET /animes?page=3
  • GET /animes/populares para traer los posters de las animes populares.
    Para navegar entre paginas usa el query ?page. ej: GET /animes/populares?page=3
  • GET /animes/estrenos para traer los posters de las animes estrenos, por el momento esta vacio.
    Para navegar entre paginas usa el query ?page. ej: GET /animes/estrenos?page=3
Obtener informacion de un Anime
  • usa /anime/:info para traer la informacion de una anime. ej: GET /anime/trese
Obtener temporadas de un Anime
  • usa /anime/temporadas/:info para traer todas las temporadas de una anime. ej: GET /anime/temporadas/trese
Obtener reproductores de un Anime
  • usa /anime/:info/temporada/:temporada/capitulo/:capitulo para traer todas los reproductores de un anime y ver las opciones de navegacion entre capitulos.
    ej: GET /anime/trese/temporada/1/capitulo/4

Buscador

Buscar una pelicula, serie o anime
  • Para buscar una pelicula usa la ruta /search mas la query ?s. ej: GET /search?s=terror
    Para navegar entre paginas usa el query ?page ej: GET /search?s=terror&page=3

Generos

Traer una lista basica de Generos
  • GET /generos para traer una lista basica de generos
Trear todos los posters de un Genero
  • /generos/:genero para traer todos los posters de un genero, de todos los tipos (peliculas, series y animes). ej GET /generos/accion
Trear todos los posters de peliculas de un Genero
  • /generos/:genero/peliculas para traer todos los posters de peliculas de un genero. ej GET /generos/accion/peliculas
Trear todos los posters de series de un Genero
  • /generos/:genero/series para traer todos los posters de series de un genero. ej GET /generos/accion/series
Trear todos los posters de animes de un Genero
  • /generos/:genero/animes para traer todos los posters de animes de un genero. ej GET /generos/accion/animes

Años

Traer una lista de todos los Años
  • GET /years para traer una lista de todos los años que contienen peliculas, series y animes
Trear todos los posters de un Año
  • /year/:year para traer todos los posters de un año, de todos los tipos (peliculas, series y animes). ej GET /year/2021
Trear todos los posters de peliculas de un Año
  • /year/:year/peliculas para traer todos los posters de peliculas de un año. ej GET /year/2021/peliculas
Trear todos los posters de series de un Año
  • /year/:year/series para traer todos los posters de series de un año. ej GET /year/2021/series
Trear todos los posters de animes de un Año
  • /year/:year/animes para traer todos los posters de animes de un año. ej GET /year/2021/animes
