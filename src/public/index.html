Peliculas

Obtener los Posters de Peliculas
  • GET /peliculas obtiene los posters de las peliculas.
    Para navegar entre paginas usa el query ?page. ej: GET /peliculas?page=3
  • GET /peliculas/populares para traer los posters de las peliculas populares.
    Para navegar entre paginas usa el query ?page. ej: GET /peliculas/populares?page=3
  • GET /peliculas/estrenos para traer los posters de las peliculas estrenos.
    Para navegar entre paginas usa el query ?page. ej: GET /peliculas/estrenos?page=3
Obtener informacion de una Pelicula
  • usa /peliculas/:info para traer la informacion de una pelicula. ej: GET /pelicula/friends-the-reunion
Obtener reproductores de una Pelicula
  • usa /peliculas/rep/:info para traer todos los reproductores de una pelicula. ej: GET /pelicula/rep/friends-the-reunion

Series

Obtener los Posters de Series
  • GET /series obtiene los posters de las series.
    Para navegar entre paginas usa el query ?page. ej: GET /series?page=3
  • GET /series/populares para traer los posters de las series populares.
    Para navegar entre paginas usa el query ?page. ej: GET /series/populares?page=3
  • GET /series/estrenos para traer los posters de las series estrenos.
    Para navegar entre paginas usa el query ?page. ej: GET /series/estrenos?page=3
Obtener informacion de una Serie
  • usa /serie/:info para traer la informacion de una serie. ej: GET /serie/the-flash
Obtener temporadas de una Serie
  • usa /serie/temporadas/:info para traer todas las temporadas de una serie. ej: GET /serie/temporadas/the-flash
Obtener reproductores de una Serie
  • usa /serie/:info/temporada/:temporada/capitulo/:capitulo para traer todas los reproductores de una serie y ver las opciones de navegacion entre capitulos.
    ej: GET /serie/the-flash/temporada/2/capitulo/16

Obtener temporadas de un Anime

Obtener reproductores de un Anime

Buscador

Buscar una pelicula, serie o anime
  • Para buscar una pelicula usa la ruta /search mas la query ?s. ej: GET /search?s=terror
    Para navegar entre paginas usa el query ?page ej: GET /search?s=terror&page=3

Generos

Trear todos los posters de un Genero

Trear todos los posters de series de un Genero

Trear todos los posters de animes de un Genero
